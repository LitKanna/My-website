# Peptide Source Website

This repository contains a simple starting point for recreating the [Peptide Source](https://peptidesource.com.au/) website. The current content is a minimal placeholder.

## Getting Started

1. Clone the repository:
   ```bash
   git clone <REPOSITORY_URL>
   ```
2. Install any dependencies if you expand beyond static HTML.
3. Open `index.html` in your browser to see the placeholder page.

## Granting Access

To allow contributors to work on this site:

1. Create a repository on GitHub (or your preferred Git host).
2. Add your collaborators under the repository settings.
3. Push the contents of this folder to the remote repository using:
   ```bash
   git remote add origin <REPOSITORY_URL>
   git push -u origin main
   ```

From there other contributors can clone the repo and submit pull requests.
